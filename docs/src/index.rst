TLDR
----

Install Piccolo:

.. code-block:: bash

   pip install piccolo


Experiment with queries:

.. code-block:: bash

   piccolo playground run


Give me an ASGI web app!

.. code-block:: bash

   piccolo asgi new

FastAPI, Starlette, and BlackSheep are currently supported, with more coming
soon.
